# ❤️ ImpaX – Donation Management Platform

## Overview

ImpaX is an end-to-end donation platform designed to connect donors, NGOs, and charitable organizations through a transparent and user-friendly experience.

The platform enables users to discover causes, donate securely, manage donations, and provides organizations with powerful dashboards to manage campaigns, documents, reports, and donor engagement.

---

# Problem

Many donation platforms lack transparency and provide poor user experiences for both donors and organizations.

ImpaX was designed to simplify the donation journey while giving organizations better tools to manage operations.
